1. أساسيات البرمجة ومقدمة في C#

- ما هي البرمجة ولماذا تختار C#؟

البرمجة هي عملية كتابة تعليمات تُنفذها الحاسوب لحل مشكلة معينة.
C# هي لغة قوية، طورتها مايكروسوفت، وتستخدم لتطوير تطبيقات سطح المكتب، المواقع الإلكترونية، وحتى الألعاب. مميزاتها:

  • سهلة التعلم.
  • مدعومة بتقنيات حديثة مثل .NET.
  • مناسبة للتطبيقات الكبيرة والصغيرة.

- أدوات البرمجة (IDE)

  1. Visual Studio: بيئة متكاملة بها كل ما تحتاجه لتطوير التطبيقات.
  2. Visual Studio Code: محرر خفيف ومثالي للمشاريع الصغيرة.

- كتابة أول برنامج (Hello World)

csharp
using System;

class Program
{
static void Main()
{
    Console.WriteLine("Hello, World!");
}
}
  • using System;: تضمين مكتبة أساسية.
  • Console.WriteLine: طباعة النص على الشاشة.

2. التحكم في تدفق البرنامج

- الجمل الشرطية (if, else)

تستخدم لاتخاذ قرارات بناءً على شروط.

csharp
int age = 20;
if (age >= 18)
{
Console.WriteLine("You are an adult.");
}
else
{
Console.WriteLine("You are a minor.");
}

- الحلقات التكرارية

مثال على استخدام for لطباعة الأرقام من 1 إلى 5:

csharp
for (int i = 1; i <= 5; i++)
{
Console.WriteLine(i);
}

foreach مناسبة للتعامل مع المجموعات:

csharp
int[] numbers = { 1, 2, 3, 4 };
foreach (int number in numbers)
{
Console.WriteLine(number);
}

3. المصفوفات (Arrays) والمجموعات (Collections)

- المصفوفات

هي مجموعة عناصر من نفس النوع.

csharp
int[] numbers = { 10, 20, 30 };
Console.WriteLine(numbers[0]); // Output: 10

- القوائم (Lists)

قوائم ديناميكية تسمح بإضافة العناصر أو إزالتها بسهولة.

csharp
List<int> numbers = new List<int>();
numbers.Add(10);
numbers.Add(20);
Console.WriteLine(numbers.Count); // Output: 2

4. الدوال (Functions)

الدوال تساعد في تقسيم الكود لقطع صغيرة لإعادة الاستخدام.

csharp
static int Add(int a, int b)
{
return a + b;
}

Console.WriteLine(Add(5, 3)); // Output: 8

5. البرمجة الكائنية (OOP)

- الكائنات (Objects) والفئات (Classes)

csharp
class Car
{
public string Brand;
public void Drive()
{
    Console.WriteLine("Driving...");
}
}

Car myCar = new Car();
myCar.Brand = "Toyota";
myCar.Drive();

- الوراثة (Inheritance)

تعني إعادة استخدام الكود من فئة رئيسية.

csharp
class Vehicle
{
public void Move()
{
    Console.WriteLine("Moving...");
}
}

class Car : Vehicle
{
public void Honk()
{
    Console.WriteLine("Honking...");
}
}

6. التعامل مع الأخطاء (Exception Handling)

لحماية البرنامج من الأعطال المفاجئة.

csharp
try
{
int result = 10 / 0; // خطأ
}
catch (DivideByZeroException)
{
Console.WriteLine("Cannot divide by zero.");
}
finally
{
Console.WriteLine("Done.");
}

7. العمل مع الملفات (File Handling)

- كتابة ملف:

csharp
File.WriteAllText("example.txt", "Hello, File!");

- قراءة ملف:

csharp
string content = File.ReadAllText("example.txt");
Console.WriteLine(content);

8. البرمجة المتزامنة (Asynchronous Programming)

لتنفيذ العمليات الكبيرة دون تجميد البرنامج.

csharp
async Task LoadData()
{
await Task.Delay(1000); // محاكاة انتظار
Console.WriteLine("Data loaded.");
}

9. المشاريع العملية والتطبيقات

  • تطبيق Console: تطبيق آلة حاسبة يقوم بعمليات الجمع والطرح.
  • تطبيق Windows Forms: بناء نافذة تسجيل دخول بسيطة.
  • تطبيق Web: تطوير واجهة API تعرض بيانات بسيطة.

10. المواضيع المتقدمة

- LINQ:

طريقة حديثة للاستعلام عن البيانات.

csharp
int[] numbers = { 1, 2, 3, 4 };
var evenNumbers = numbers.Where(n => n % 2 == 0);
foreach (var number in evenNumbers)
{
Console.WriteLine(number); // Output: 2, 4
}

- Entity Framework:

أداة تربط بين الكود وقواعد البيانات.
